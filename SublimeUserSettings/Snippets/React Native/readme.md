# Snippets

> These work only in jsx/tsx files

## Normal

| Trigger | Description |
| ------- | ----------- |
| s       | style={{|}} |
| ss      | StyleSheet  |
| set     | useState    |
| T       | Text        |
| V       | View        |

| fl      | flex                   |

| fs      | fontSize               |
| h       | height                 |
| jc      | justifyContent         |

| mv      | marginVertical         |

| pv      | paddingVertical        |
